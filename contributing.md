# Contributing to TypeVocab

First off, thank you for considering contributing to TypeVocab! This project is designed to be beginner-friendly. We especially welcome students who are making their first Open Source contributions.

## How Can I Contribute?

The easiest way to contribute is by adding new Kazakh words to our vocabulary database. 

### Step-by-Step Guide to Adding Words

1. **Fork the Repository:** Click the "Fork" button at the top right of this page to create a copy of this repository in your own GitHub account.
2. **Clone your Fork:**
