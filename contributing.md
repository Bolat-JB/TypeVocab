# Contributing to TypeVocab

First off, thank you for considering contributing to TypeVocab! 🎉 

TypeVocab is an open-source, Next.js-based application designed to make learning Kazakh vocabulary fast, free, and accessible. This document provides guidelines to ensure a smooth collaborative process.
